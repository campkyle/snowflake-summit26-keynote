Snap and Ask
  • 화면의 어느 영역이든 캡처하여 CoCo에게 자연어로 질문
  • 차트, 대시보드, 오류 메시지, 코드 스크린샷 등 모두 지원
  • CoCo가 시각적 컨텍스트를 완전히 이해하고 분석 결과 제공

  • 개발자·분석가 모두 사용 가능, 코드 없이도 인사이트 도출

  2. Snap and Ask로 해당 영역 캡처
  3. "왜 지난주에 비해 30% 하락했나요?" 질문
  4. CoCo가 관련 테이블·파이프라인을 조사해 원인 제시

ACT 2 · 신뢰
Governance & Security — Trust as Accelerant
  • Intent-driven Governance: "모든 PII를 보호해줘" → 자동 분류·정책 생성 (CoCo 기반)
  • Agent Identity ↗ — 에이전트 실행 컨텍스트 감지, 마스킹 정책에서 참조
  • Data Movement Policies: 태그된 데이터의 외부 전송·다운로드 시스템 레벨 차단
  • Multi-party Approvals (PP): 고위험 작업은 2인 승인 필수
  • Horizon Context (NEW) ↗ — 에이전트에 시맨틱 신호 자동 제공
  • Interactive Compiler: 40배 빠른 컴파일 → 3~4배 워크로드 가속
  • Adaptive Compute GA: 1세대 대비 약 2배 성능
  • AI 비용 거버넌스: AI 사용량을 Budget에 통합, 사용자별 쿼터

ACT 2 · 상세
Agent Identity & Horizon Context
🔐 Agent Identity
  • 에이전트 실행 컨텍스트를 감지하는 새로운 컨텍스트 함수
  • 마스킹 정책 및 행 접근 정책에서 직접 참조 가능
  • 에이전트 유형별로 다른 데이터 접근 권한 설정
  • "전통적 보안 모델은 인간 사용자를 위해 설계되었습니다. 에이전트는 다릅니다."
🧠 Horizon Context (NEW)
  • CoCo·CoWork·Cortex 에이전트에 시맨틱 신호를 자동으로 제공
  • Semantic Views와 외부 메타데이터 커넥터 통합
  • 에이전트가 비즈니스 컨텍스트를 이해하고 더 정확한 답변 제공
  • Cortex Sense와 연계 ↗ — 정확도 24% → 83%
Horizon AI Guardrails
  • CoCo/CoWork를 탈옥 시도로부터 런타임에서 보호
  • 프롬프트 인젝션 공격 방어 (기존 + 제로데이)
Interactive Compiler
  • 새로운 쿼리 컴파일러 도입 — 컴파일 속도 40배 향상
  • 전체 워크로드 3~4배 가속

ACT 3 · 개방성
Open & Interoperable — No Silos
  • Apache Iceberg V3 GA: 가장 광범위한 구현. V4 스펙 개발 참여. 완전한 양방향 카탈로그 상호운용
  • Snowflake Managed Iceberg Storage GA: AWS + Azure GA (GCP 예정). 벤더 종속 없음
  • SAP Zero-copy 통합 GA: 가장 많이 요청된 파트너 통합
  • Workday Data Cloud · IBM Watsonx · AVEVA Connect 추가 Zero-copy
  • Open Sharing (PP): Iceberg REST로 비Snowflake 사용자와 데이터 공유
  • Multi-party Collaboration GA: 여러 조직이 하나의 안전한 환경에서 협업. Netflix 얼리 어답터
  • Data Re-sharing GA: 수신한 데이터를 제3자에게 자유롭게 재공유
  • Open Semantic Interchange: Snowflake 창립, 시맨틱 레이어 개방형 표준화
