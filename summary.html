Day 2 플랫폼 키노트는 Snowflake 공동창업자 Benoit Dageville이 창업 10주년을 돌아보며 시작되었습니다. 이후 제품 EVP Christian Kleinerman이 90분에 걸쳐 30개 이상의 제품 발표를 진행했습니다. 세션의 첫 번째 하이라이트는 공식 명칭 변경 — Cortex Code → Snowflake CoCo, Snowflake Intelligence → Snowflake CoWork — 였으며, 관중으로부터 강렬한 반응을 이끌어냈습니다.

키노트는 4막 구조로 진행되었습니다: 사용 편의성 / 신뢰 / 개방성 & 상호운용성 / 모두를 위한 CoWork. 주요 발표로는 CoCo Desktop GA, Kafka 호환 스트리밍 Snowflake Data Stream, 에이전트 정확도를 획기적으로 향상시킨 Cortex Sense, SAP·Workday 등과의 제로카피 통합 확장, Adaptive Compute GA(약 2배 빠른 성능) 등이 있었습니다.

Thomson Reuters(거버넌스), Under Armour(상호운용성), Samsung(CoWork)이 무대에 올라 실제 운영 환경에서 AI를 활용하는 방법을 공유했습니다. 키노트의 마지막 메시지는 "Snowflake는 여러분을 'can we(할 수 있을까?)' 시대에서 'shall we(이제 해볼까?)' 시대로 이끕니다."였습니다.

Act 1: 사용 편의성 — 마찰 제거
23:27 – 50:00
  • CoCo Desktop GA: Plan → Build → Deploy를 하나의 흐름으로. CLI 수준의 파워를 UI로.
  • Cloud Agents (nearly GA): Snowsight과 CLI에 샌드박스 추가. 예약 실행, Async API, Skill Catalog 지원.
  • Snap and Ask: 화면 어느 부분이든 캡처해 CoCo에게 질문 가능. 완전한 시각적 컨텍스트 제공.
  • Snowflake Data Stream (Private Preview 예정): 완전 관리형 Kafka Wire 호환 스트리밍. 제로카피, 밀리초 미만의 지연시간.
  • AIM (AI 기반 마이그레이션): Teradata 가상화 지원 (Snowflake에서 Teradata SQL 그대로 실행). Spark RDD → Snowflake 5배 성능 향상.
  • Cortex Function Studio (Public Preview): 맞춤형 AI 함수 빌드·평가·롤아웃.
  • SpaceX Grok 모델이 Cortex에 추가 (Private Preview).
  • Agentic Search (Public Preview): 비정형 데이터에 대한 정밀한 분석 쿼리 — RAG가 아닌 구조화 변환 → SQL 응답.
  • Snowflake App Runtime (Public Preview): Snowflake 내에서 Node.js/React 앱 호스팅. 단일 snow app deploy 명령으로 배포.
  • Streamlit in Snowflake GA: 워크스페이스, Git 통합, 컨테이너 실행. 월 활성 개발자 170만 명.

Act 2: 신뢰 — 거버넌스 & 보안
50:00 – 1:14:15
  • 의도 기반 거버넌스: "모든 PII를 보호해줘"라고 말하면 자동 분류·정책 생성·지속적 거버넌스 시작 (CoCo/AI 기반).
  • Horizon AI Guardrails: CoCo/CoWork를 런타임에서 탈옥 시도와 프롬프트 인젝션으로부터 보호.
  • Agent Identity: 에이전트 실행 컨텍스트를 감지하는 새로운 컨텍스트 함수. 마스킹 정책과 행 접근 정책에서 참조 가능.
  • Data Movement Policies: 태그된 데이터를 외부 스테이지로 전송하거나 Snowsight을 통해 다운로드하는 것을 차단.
  • Multi-party Approvals (Private Preview): 고위험 작업은 2인 승인 필요. 단독 관리자도 MFA를 일방적으로 비활성화할 수 없음.
  • Horizon Context (NEW): CoCo/CoWork/Cortex 에이전트에 시맨틱 신호를 자동으로 제공. Semantic Views와 외부 메타데이터 커넥터 통합.
  • Adaptive Compute GA: 1세대 대비 약 2배 빠름, 현재 롤아웃 중.
  • Interactive Compiler (NEW): 새로운 쿼리 컴파일러로 40배 빠른 컴파일 → 3~4배의 워크로드 가속.
  • Unistore 8배 최적화 (Public Preview): 지연시간과 처리량 획기적 개선. Hybrid Table 사용자 재평가 권장.
  • AI 비용 거버넌스: AI 사용량을 Budget에 통합. 사용자별 쿼터, Budget 커스텀 액션 (임계치에서 저장 프로시저 트리거).
  • Observe 인수 완료: 로그, APM, 인프라 모니터링을 Snowflake에 네이티브로 통합. CoCo CLI + Iceberg 호환.

Act 4: 모두를 위한 CoWork — 모두의 제어 플레인
1:28:15 – 1:47:15

지금 바로

CoCo Desktop을 설치(현재 GA)하고 자연어로 파이프라인을 빌드·수정하는 경험을 해보세요. CLI, Snowsight, Desktop 중 워크플로우에 맞는 형태를 선택하면 됩니다. 파괴적 작업은 항상 사용자 승인이 필요하므로 안전하게 탐색할 수 있습니다.

지금 바로

Snowflake CoWork(구 Snowflake Intelligence)를 배포하여 조직 내 모든 사람이 개인 AI 어시스턴트를 갖도록 하세요. Personal Work Engine이 에이전트 선택을 없애고, Cortex Sense가 자동으로 정확도를 24% → 83%로 높여줍니다.

곧 시작

AI를 운영에 배포하기 전에 거버넌스를 먼저 구축하세요. 의도 기반 거버넌스(자연어로 정책), Agent Identity, Data Movement Policy를 결합하세요 — Thomson Reuters의 "거버넌스가 가속 장치" 모델을 따라하세요.

곧 시작

SAP, Workday, Salesforce와의 제로카피 통합을 평가하거나, Open Sharing (Iceberg REST)으로 비Snowflake 엔진에 데이터를 제공하세요. Snowflake Managed Iceberg Storage는 AWS와 Azure에서 현재 GA입니다.

곧 시작

Kafka를 사용 중이라면 Snowflake Data Stream(Kafka Wire 호환, 완전 관리형, 제로카피, 밀리초 미만 지연시간)으로 전환하는 비용 절감 가능성을 평가해 보세요. Private Preview 신청이 열려 있습니다.
